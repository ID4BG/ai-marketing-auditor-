## AI Marketing Auditor

Minimal SaaS-style landing page built with **Next.js 14 (App Router)** and **TailwindCSS**.

- **Title**: AI Marketing Auditor  
- **Inputs**: Website URL, LinkedIn page URL, ad copy textarea  
- **Action**: "Analyze My Marketing" button posts to `/api/analyze`  
- **Output sections**: Positioning, Messaging, ICP Clarity, Funnel Analysis

### Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.
